JustClover — полный UI редизайн

Что внутри:
- index.html
- style.css
- app.js
- service-worker.js
- папка themes/ с превью для раздела «Оформление»

Как поставить:
1. Сделай бэкап текущих файлов сайта.
2. Скопируй содержимое этого архива в корень проекта с заменой файлов.
3. Задеплой на GitHub Pages / свой хостинг.
4. После обновления открой сайт и сделай жёсткое обновление страницы (Ctrl+F5).

Что изменено:
- новый layout страницы просмотра;
- левый rail и центральный плеер в стиле выбранного макета;
- правый столбец: участники, голос, чат;
- более живой, но не вырвиглазный анимированный фон;
- раздел «Оформление» с несколькими вариантами по прошлым генерациям.
